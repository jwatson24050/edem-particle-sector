# Papers

This directory holds the two companion papers of the EDEM particle-sector
project.

| File | arXiv class | Contents |
|---|---|---|
| `EDEM_Paper_A_Note_v2.docx` | math.NT / math.CO | Pure-mathematics note: the σ-lattice, the Φ₆-tower, and four theorems about their intersection. No physical claims. |
| `EDEM_Paper_B_v74.docx` | hep-ph / hep-th | Physics paper: maps the arithmetic of Paper A to the Standard Model under five physical postulates (P1–P5). Derives the charged-fermion masses, the electroweak and strong couplings, and the PMNS angles. |

The `.docx` files are the **current working versions** of the manuscripts.
LaTeX / arXiv-ready versions are in preparation and will be added here when
available.

For the project overview, the key-predictions table, the postulates, and the
open-problems list, see the [root README](../README.md). The computational
checks of the PMNS results live in [`../verification/`](../verification/).
